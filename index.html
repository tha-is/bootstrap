Instalando o bootstrap por npm

  • crie uma pasta para salvar seu pacote;
  • abra a pasta no ide;
  • abra o terminal e: npm init;
  • coloque o nome do projeto e dê enter até a pergunta final: Is this OK?;
  • ao finalizar o projeto digite: install bootstrap

    • Instalando icons

  • abra a pasta do projeto
  • abra a pasta no ide;
  • abra o terminal e: install bootstrap-icons;
  • no seu head do seu index.html acrescente o seguinte código:
    link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css
  • para testar use o icon de coração utilizado nesse template com o seguinte código:
    i class="bi bi-heart
